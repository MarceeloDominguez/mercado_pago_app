## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/mercado_pago_app.git
```

Go to the project directory

```bash
  cd mercado_pago_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
